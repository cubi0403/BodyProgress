# BodyProgress

BodyProgress is an open-source fitness coaching analytics tool.

## Features

* Track body weight
* Track body fat percentage
* Track muscle mass
* Track visceral fat
* Track body age
* Track body measurements

## Analytics

* Weight change
* Body fat change
* Muscle gain/loss
* Total body measurement change
* Coach summary report

## Tech Stack

* Python
* CSV database

## Future Roadmap

* Charts and trends
* PDF reports
* Web dashboard
